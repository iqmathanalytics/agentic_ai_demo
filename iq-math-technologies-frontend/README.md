# IQ Math Technologies Frontend

React landing page inspired by the course-site structure at `nexpertsai.com`, adapted for IQ Math Technologies with India-focused copy and INR pricing.

## Run locally

Install dependencies:

```powershell
npm.cmd install
```

Start the Vite dev server:

```powershell
npm.cmd run dev
```

## Build

```powershell
npm.cmd run build
```

Preview the production build:

```powershell
npm.cmd run preview
```
